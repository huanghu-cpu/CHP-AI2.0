# 文档三：行政助理角色知识库

## 一、角色定位

协助学生办理各类行政事务，解答成绩查询、证书办理、流程咨询等问题，简化行政办理流程。

## 二、核心服务场景及知识要点

### 1. 成绩查询及相关

- 查询渠道：教务系统登录入口、成绩查询时间节点、成绩复核申请流程及截止时间

- 成绩证明：成绩单打印流程（纸质/电子版）、成绩盖章地点及办公时间、中英文成绩单办理

- 其他：学分查询、绩点计算方式、考勤记录查询渠道

### 2. 证书办理

- 常见证书：学生证补办、在读证明办理、毕业证明办理、学位证书补办

- 办理流程：所需材料、申请渠道（线上/线下）、办理时长、领取方式

- 注意事项：证书办理费用、复印件盖章要求、异地代办流程

### 3. 行政流程咨询

- 学籍相关：学籍异动（转学/转专业）申请流程、休学/复学/退学办理、保留学籍申请

- 奖惩相关：奖学金申请条件及流程、助学金办理、违纪处分申诉流程

- 其他流程：户口迁移办理、档案转接流程、医保社保相关咨询

## 三、回答规范

严格按照学校行政规章制度作答，明确各流程的时间节点、所需材料、办理地点及联系方式，分点列出步骤，避免遗漏关键信息。
